Overview

프로젝트 소개

Rhythm Graph는 그래프가 만들어 내는 형태적 리듬과 리듬 게임 특유의 타이밍 플레이를 결합한 프로젝트입니다. 플레이어는 화면에 나타나는 흐름을 읽고 적절한 순간에 반응해야 하며, 시각적 패턴과 입력 감각이 함께 맞물릴 때 더 높은 몰입을 경험할 수 있습니다.

Core Features

게임 특징

🎵

타이밍 중심 플레이

정확한 입력 타이밍이 핵심입니다. 리듬을 익히고 판정 구간에 적응할수록 더 안정적인 콤보와 높은 점수를 노릴 수 있습니다.

📈

그래프 기반 비주얼

사인, 코사인, 탄젠트 등 그래프의 흐름을 시각적 테마와 플레이 감각에 반영해, 일반적인 리듬 게임과는 다른 분위기를 만듭니다.

✨

몰입형 연출

네온 톤의 컬러, 부드러운 움직임, 간결한 UI 구성을 통해 차분하면서도 선명한 플레이 경험을 지향합니다.

How to Play

플레이 가이드

화면에 나타나는 흐름과 입력 포인트를 확인하고, 가장 적절한 순간에 반응하세요. 그래프의 움직임을 먼저 읽고 타이밍을 안정적으로 맞추는 것이 고득점의 핵심입니다.

플레이 포인트

  • 그래프의 흐름을 먼저 파악하기
  • 입력 타이밍을 일정하게 유지하기
  • 미스를 줄여 콤보를 이어 가기
  • 모드별 시각적 차이를 감각적으로 익히기

앞으로의 확장 방향

  • 추가 음악과 배경 테마
  • 더 다양한 그래프 패턴
  • 모드 및 난이도 확장
  • 업데이트 로그와 안내 개선
Project Notes

AI로 만든 취미 프로젝트

Rhythm Graph는 상업적 서비스보다는 아이디어를 실제 결과물로 구현해 보는 과정 자체에 의미를 둔 개인 프로젝트입니다. 개발 과정에서는 AI를 적극적으로 활용해 디자인, 코드 작성, 구조 정리, 문구 개선 등 제작 전반의 완성도를 높였습니다.

AI Hobby Project

기획 아이디어를 제외한 대부분의 제작 과정은 AI의 도움을 바탕으로 완성했습니다.

이 프로젝트는 작은 개인 작업이지만, 새로운 방식의 게임 감각을 실험하고 빠르게 형태로 옮기는 데 집중했습니다. 완벽한 상용 제품보다는, 독특하고 인상적인 결과물을 만드는 과정에 더 큰 가치를 두고 있습니다.

Details
제작 방식 방향성과 핵심 아이디어는 직접 정하고, 구현과 정리 과정에서는 AI를 활용해 속도와 완성도를 끌어올렸습니다.
프로젝트 성격 개인적인 흥미와 실험에서 출발한 취미 프로젝트이므로, 구조와 연출은 앞으로도 자유롭게 바뀔 수 있습니다.
업데이트 스타일 정해진 상용 개발 일정 대신, 새로운 아이디어가 생길 때마다 천천히 개선해 나가는 방식으로 이어집니다.
FAQ

자주 묻는 질문

어디까지 AI로 만들었나요?
프로젝트의 핵심 아이디어와 방향성은 직접 정했습니다. 다만 구현, 디자인 정리, 문구 다듬기, 구조 개선 등 기획을 제외한 대부분의 제작 과정에는 AI를 적극적으로 활용했습니다.
왜 무료인가요?
Rhythm Graph는 수익화를 목적으로 운영하는 서비스가 아니라, 실험과 재미를 중심으로 만든 개인 취미 프로젝트이기 때문입니다. 누구나 부담 없이 경험할 수 있도록 무료로 공개하고 있습니다.
업데이트는 규칙적인가요?
규칙적인 편은 아닙니다. 이 프로젝트는 취미로 진행되는 작업이기 때문에, 업데이트 주기는 개인 일정과 새로운 아이디어에 따라 유동적으로 달라질 수 있습니다.
